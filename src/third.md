# 一个持久化的单链栈

好了，我们已经掌握了可变单链堆栈的艺术。

让我们通过编写一个*持久化的*不可变的单链表，从*单一*所有权转向*共享*所有权。这
将是函数式程序员所熟悉和喜爱的列表。你可以得到头或尾，把别人的头放在别人的尾
上......而且......基本上就是这样了。不变性是一种地狱般的毒药。

在这个过程中，我们很大程度上只是熟悉了Rc和Arc，但这将为我们下一个*改变游戏*的链
表做准备。

让我们添加一个名为`third.rs`的新文件：

```rust ,ignore
// in lib.rs
```
